This project started during my post-semester holidays with a simple conversation in the kitchen:

Me: "Mom, I've got some free time. Want me to make something for you?"
Mom: "Actually, yes! It would be helpful to have phone-controlled switches for the tube light, the LED bulb, and the two fans. So I can turn them off from here or when I'm reading in bed."
Me: "Like the setup the neighbors just got?"
Mom: "Exactly. I actually called them to ask for a price, and they quoted ₹2,000 for each switch and ₹1,000 for the hub!"
Me: "That’s daylight robbery! Don't pay that, I can build this."

Commercial smart home setups in India can easily cost upwards of ₹9,000 for a single room. This project was born out of a desire to provide the exact same utility at the absolute cheapest possible price, without any compromises on stability or speed.
